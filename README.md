# RAM Digital Performance Dashboard

Ready-to-deploy one-page dashboard for Vercel.

## Upload to GitHub
1. Create a new GitHub repository.
2. Upload these files: `index.html`, `package.json`, `README.md`.
3. Commit changes.

## Deploy on Vercel
1. Open Vercel > Add New Project.
2. Import the GitHub repository.
3. Keep default settings.
4. Click Deploy.

The dashboard uses RAM identity colors: green, brown, gold and cream.
